# FTF Frequently Asked Questions

Common questions about Free Tools Friday (#FTF) by Peach Security.

---

## General

### What is Free Tools Friday?

Free Tools Friday (#FTF) is Peach Security's initiative to release useful MSP tools, resources, and guides every Friday. No hoops. No paywalls.

### Where can I read the full manifesto?

The official manifesto is published at: https://www.peachsecurity.io/ftf

### Who is FTF for?

FTF is designed for MSPs (Managed Service Providers) who need practical, useful tools without the typical industry friction.

---

## Tools

### Where do tools live?

Tools with real code live in their own repositories under the [Peach-Security GitHub organization](https://github.com/Peach-Security). This repo serves as the official index.

### How do I use a tool?

Each tool repository contains its own README with installation and usage instructions.
